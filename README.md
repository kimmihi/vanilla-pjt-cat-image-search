# 고양이 사진 검색 사이트

### Intro

프로그래머스 과제에 있는 프로젝트이며, 라이브러리 및 프레임워크을 도입하지 않고 순수 자바스크립트, HTML, CSS를 사용하여 진행한다.
해당 과제를 통해 자주쓰는 라이브러리 및 프레임워크가 어떤 문제를 해결해주는지 이해하고, 명세서를 보면서 평소에는 생각해보지 못했던 것들을 해결해보려고 한다.
